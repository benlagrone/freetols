# Thumbnail Wizard Roadmap

T-shirt sizing in this roadmap:
- `S`: 0.5-1.5 days
- `M`: 2-4 days
- `L`: 1-2 weeks
- `XL`: 2-4+ weeks

## Priority Order

1. Keyboard shortcuts (`S`)
2. History panel: undo/redo + snapshots (`M`)
3. Layers panel: reorder, lock/hide, group/ungroup (`L`)
4. Guides/rulers/snapping/safe zones (`M`)
5. Crop tool with common aspect ratios (`M`)
6. Free transform: skew/perspective/warp (`L`)
7. Layer effects: stroke/shadow/glow (`M`)
8. Blend modes (`M`)
9. Text effects: outlines + curved text (`M`)
10. Filters v1: blur/sharpen/noise/pixelate (`M`)
11. Brush + eraser toolset (`L`)
12. Marquee + lasso selection (`L`)
13. Magic wand / color range select (`XL`)
14. Adjustment layers: levels/curves/hue-sat (`XL`)
15. Clone stamp + healing brush (`XL`)

## Immediate Implementation Scope

Completed:
- Item 1: Keyboard shortcuts
- Item 2: History panel (undo/redo + snapshots)
- Item 3: Layers panel (reorder, lock/hide, group/ungroup)
- Item 4: Guides/rulers/snapping/safe zones
